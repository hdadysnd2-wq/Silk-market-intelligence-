## 1 · ما الذي يعطيه الباك-إند اليوم · What the backend delivers today

### ✅ مجاني (جاهز)
ترتيب الأسواق (`silk_market_ranker.py`) · حجم الاستيراد وحصة السعودية والدول المورّدة وحصصها
(`silk_agents.py:107,175`) · دخل الفرد PPP والسكان · تركّز المنافسة HHI · الاشتراطات (دخول/خروج،
`silk_requirements_agent.py`) · **مدى السنوات وخط الاتجاه والنمو/CAGR** (`silk_trend.py` — جديد) ·
الحكم والثقة (`silk_synthesis.py`) · **مؤشر اكتمال الدراسة** (`silk_render._completeness` — جديد).

### 🟡 يحتاج مُدخَلاً منك و/أو تعميقاً مدفوعاً
موقعك وهامشك مقابل المنافسين (`correlation.py` → `feasibility_threads`، يحتاج بطاقة المنتج) ·
مقارنة سعرك (`own_price` → `price_comparison`) · **أسعار الرفّ الفعلية** (`silk_localprice_agent.py`
— مدفوع عبر `/deepen`) · المستوردون بالاسم (Volza/explee).

### ملاحظة مبدئية
حيثما تغيب البيانات، تعرض المنصّة **«غير مرصود»** لا رقماً مخترعاً — وهذا يسري على الواجهة الجديدة
أيضاً (تحقّق حيّ في المتصفح: خانة الاتجاه ومقارنة الأسعار تظهران «غير مرصود» بلا شبكة).

## 2 · العقدة الحاسمة: أسعار المنتجات بدقّة · The critical node: accurate prices

**لا يوجد API رسمي مجاني نظيف لأسعار التجزئة — هذا واقع الصناعة.** المُحوِّل جاهز
(`silk_localprice_agent.py` يقبل أي مزوّد عبر `LOCALPRICE_API_URL`)؛ القرار **تجاري لا برمجي**.

| المزوّد · Provider | التغطية · Coverage | الدقّة | التكلفة التقريبية | ملاحظة صريحة |
|---|---|---|---|---|
| **Keepa** | أمازون (تاريخ أسعار) | عالية جداً | ~€19–49/شهر | الأفضل لأمازون؛ تاريخ أسعار غني |
| **Rainforest API** | أمازون (فوري) | عالية | ~$50+/شهر حسب الحجم | جاهز، بلا بنية كشط |
| **SerpApi (Google Shopping)** | متعدّد المتاجر | متوسطة-عالية | ~$50–130/شهر | الافتراضي الحالي في المُحوِّل |
| **Bright Data / Oxylabs / Apify** | نون، جوميا، كارفور الخليج، أي متجر | متغيّرة | متغيّرة (بالاستخدام) | الطريق الواقعي لأسواق الخليج (لا API عام لنون/جوميا) |
| كشط ذاتي مباشر | أي متجر | هشّ | «مجاني» ظاهرياً | **غير موصى به كأساس** — ToS/صيانة/موثوقية |

**التوصية:** ابدأ بـ**Keepa (أمازون)** كبداية رخيصة دقيقة، ثم أضف **Bright Data/Apify** لنون/جوميا.
اربطهما خلف `LOCALPRICE_API_URL` مع بقاء انضباط النسب (كل سعر بمصدره وتاريخه)، و**توحيد الوحدة/العملة**
قبل المقارنة بمنتجك. كن صريحاً: تكلفة شهرية، وتغطية متفاوتة بالسوق والمنتج.

> **تحذير مبدئي:** لا تُدخِل تحويل عملات مُقدَّراً بلا مصدر سعر صرف حقيقي — ذلك اختلاق. أضِف مصدر FX
> موسوماً (مثل exchangerate.host/ECB) أو اعرض الأسعار بعملتها الأصلية كما اليوم.

## 4 · اقتراحات تطويرية مهمّة · High-value additions

1. **مقارنة عدّة أسواق جنباً لجنب** (لا سوق واحد كل مرة).
2. **محفظة منتجاتك** — احفظ منتجاتك وبدّل بينها وقارن أيّها أنسب لأي سوق (الواجهة الجديدة تحفظ بطاقة
   منتج واحدة في `localStorage`؛ التوسيع لمحفظة مباشر).
3. **تكلفة الشحن الحقيقية** لإغلاق فجوة القرب اللوجستي المعلنة (`silk_discovery.py:183`).
4. **توحيد العملات والوحدات** — ضروري لصحّة الهوامش عبر الأسواق.
5. **تنبيهات ومراقبة** عند تغيّر حصة السعودية أو نمو سوق (لا لقطة).
6. **سيناريو «ماذا لو»** على سعرك → هامشك فوري (الواجهة تحسب الهامش من بطاقة المنتج بالفعل).

## 5 · الخلاصة · Bottom line

المنصّة **مبنية جيداً، والوكلاء يعملون، وتعطي دراسة فرز حقيقية مجاناً اليوم** — وهذا الفرع أضاف
**مدى السنوات ومؤشر الاكتمال وواجهة مؤسسية ثنائية اللغة** فعلياً وباختبارات. لتصبح **أداة قرار كاملة**،
العقدة الوحيدة الحقيقية هي **ميزانية بيانات الأسعار** (الموجة ب) — وهذا قرار تجاري، والكود جاهز لاستقباله.

The platform is well-built, the agents run, and it already delivers a real free shortlist study.
This branch shipped the year-range, completeness indicator, and bilingual enterprise UI with tests.
The only real remaining node to a full decision tool is a **pricing-data budget** (wave B) — a
commercial decision the code is already wired to accept.
