# نظام تصميم Claude لسِلك · Silk Claude Design System

> نظام بصري هادئ للوحة ذكاء الأسواق. يحترم النظام القائم (IBM Plex Arabic/Mono، RTL، النِسب أولاً)
> ويهدّئ حوله: لوح محايد دافئ، accent مكتوم واحد، فراغ متوازن، حركة لطيفة. فاتح وداكن.
> عرض حيّ: `ui-preview.html`.

## 1 · الفلسفة · Philosophy
هويّة سِلك هي **المصداقية**: كل رقم بمصدره، وكل فجوة معلنة. التصميم يخدم ذلك بثلاث قواعد:
1. **القرار يهيمن** (Apple HIG: الوضوح) — الحكم وثقته أكبر عنصر؛ الباقي يذعن.
2. **النِسب تهمس** (Claude Design: الهدوء) — «المصدر · ثقة · تاريخ» نص ثانوي مكتوم لا يزاحم الرقم.
3. **الفراغ يفصل، لا الحدود** (NN/g #8) — سلّم فراغ ثابت وظل واحد خفيف بدل الصناديق المتداخلة.

## 2 · الرموز · Design Tokens

### الألوان — فاتح · Color (light)
| Token | Hex | الاستخدام |
|---|---|---|
| `--paper` | `#FAFAF8` | الخلفية (محايد دافئ منحاز للـaccent، لا رمادي ميت) |
| `--surface` | `#FFFFFF` | البطاقات |
| `--ink` | `#1C1D22` | النص الأساسي |
| `--muted` | `#6B6E76` | نص ثانوي |
| `--faint` | `#9A9DA5` | النِسب، التسميات |
| `--line` | `#ECEBE6` | خطوط شعرية دافئة |
| `--accent` | `#3E4CC0` | إنديغو مكتوم (أهدأ من `#2348E2` الحالي) |
| `--accent-soft` | `#EEF0FB` | خلفية accent خفيفة |

### دلالات الحالة (منفصلة عن الـaccent) · Semantic
| Token | Hex | المعنى |
|---|---|---|
| `--go` / `--go-soft` | `#3F7A55` / `#EAF2EC` | GO — إيجابي مبدئي |
| `--gap` / `--gap-soft` | `#9A7A2E` / `#F6EFDD` | فجوة معلنة / WATCH |
| `--no` / `--no-soft` | `#A6503F` / `#F6E9E5` | NO-GO / مؤجّل |

### داكن · Dark (يُعاد تعريف الرموز فقط)
`--paper:#131418` · `--surface:#1B1D23` · `--ink:#ECECEA` · `--muted:#9A9DA6` · `--faint:#71747D`
· `--line:#2A2C33` · `--accent:#8A97F0` · `--accent-soft:#23263A` · الدلالات أفتح تشبّعاً للأرضية الداكنة.

### الطباعة · Type (نحترم القائم)
- عربي/أساسي: **IBM Plex Sans Arabic** — Display 700, Title 600, Body 400/500.
- أرقام/رموز: **IBM Plex Mono** + `font-variant-numeric: tabular-nums` لكل عمود أرقام.
- سلّم: Display 30 · Title 20 · Subtitle 16 · Body 15 · Caption 13 · Micro 11.5 (line-height 1.55 للنص).
- التسميات العلوية (eyebrows) بأحرف كبيرة + `letter-spacing:.08em` — تُشفّر بنية لا زينة.

### الفراغ والشكل · Spacing & shape
- سلّم فراغ: `4 · 8 · 12 · 20 · 32 · 48` (متغيّر `--s1..--s6`).
- زوايا: `10px` عناصر، `16px` بطاقات. ظل واحد: `0 1px 2px rgba(20,20,30,.04), 0 8px 24px rgba(20,20,30,.05)`.
- حركة: `160–200ms cubic-bezier(.2,.6,.3,1)`؛ نبض واحد لحالة «جارٍ»؛ احترام `prefers-reduced-motion`.

## 3 · المكوّنات المُعاد تصميمها · Redesigned Components
| المكوّن | التغيير الجوهري · Key change |
|---|---|
| **هيرو القرار** | الحكم Display كبير + حلقة ثقة + سطر «لماذا» + مختصر؛ شريط دلالة علوي رفيع بدل حدّ ملوّن سميك |
| **خط الوكلاء** | شريط تقدّم أفقي هادئ يشتق من الردّ الفعلي؛ عقدة بحالتها (مرصود/فجوة معلنة) لا شارات ثابتة |
| **جدول الأسواق** | صفوف مفصولة بفراغ وخط شعري؛ شريط نقاط دقيق؛ رقاقة تغطية؛ الفجوة «غير مرصود» بتلميح ملاحظتها |
| **رقاقة النِسب** | «المصدر · ثقة c · تاريخ» بلون `--faint` وحجم Micro — حاضرة دائماً، هامسة دائماً |
| **تفصيل السوق** | رقم النقاط بطل واحد؛ المكوّنات الأربعة كبطاقات هادئة؛ الفجوة سطر صريح لا فراغ |
| **موقعك التنافسي / حدود التقرير** | زوج المصداقية جنباً لجنب بفراغ متساوٍ؛ نبرة تفسيرية لا تحذيرية صاخبة |

## 4 · خرائط الامتثال · Standards mapping (سريعة)
- **Apple HIG:** الوضوح (القرار بطل)، الإذعان (الكروم شفاف)، العمق (طبقات بالخلفية/الظل لا الإطار).
- **Material 3:** type scale منضبط؛ elevation بالخلفية والظل؛ حالات لمس/تركيز واضحة.
- **NN/g:** #1 حالة النظام (خط الوكلاء) · #2 لغة المجال العربية · #6 تمييز لا تذكّر (البحث) · #8 جمالية وحد أدنى.
